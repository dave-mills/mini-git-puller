# mini-git-puller

Listens for a webhook from GitHub and pulls the latest changes from a specified repository.
Requires Node.js and Express.

## Setup

1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Create a `.env` file in the root directory with the following content:
4. Replace `/path/to/your/repo` with the actual path to your local Git repository.
5. Start the server with `node index.js`.
